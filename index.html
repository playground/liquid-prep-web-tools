Liquid Prep Web Tools

To flash firmware onto your ESP32:

  • Plug the ESP32 into your computer
  • Click the CONNECT button
  • Select cu.usbserial-0001(if you're unsure which port your ESP32 is connected to, unplug the USB that is connected to the ESP32. The port it's connected to will disapear, then plug it back and the port should reappear.)
  • Click the CONNECT button, and then select Install Liquid Prep Web Tools
